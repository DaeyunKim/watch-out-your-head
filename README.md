# Getting Started


### Guides
API server : watchout-api ( Spring Boot - gradle )
동작 확인 : http://localhost:8080

WEB server : watchout-web ( React - npm )

1. npm  설치하기 !!

2. web server start  
`cd ./watchout-web`<br/>

`npm install` 을 해준 뒤 <br/> 
`npm start` 입력 <br/>
동작 확인 : http://localhost:3000
